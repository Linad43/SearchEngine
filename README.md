# SearchEngine

Простой поисковый движок на C++ с поддержкой JSON.

---

## Что делает проект

- Читает конфиг `config.json` и файлы документов.
- Строит обратный индекс слов.
- Считает релевантность документов для запросов.
- Поиск возвращает документы в порядке релевантности.
- Результаты записывает в `answers.json`.
- Индексация и поиск используют многопоточность.

---

## Файлы

- `config.json` — конфиг с настройками и списком файлов.
- `requests.json` — список поисковых запросов.
- `answers.json` — результат поиска.
- `src/` — исходники проекта.
- `include/` — заголовочные файлы.
- `tests/` — модульные тесты.
